예제를 통하여 아래와 같은 단점을 경험해 보실 수 있습니다

  • 주문, 배송, 상품 서비스가 모두 가동중이어야 데이터 조회가 됩니다.
  • 주문이력이 많을시에 모든 데이터를 조회 하기때문에 시간이 많이 걸립니다.
